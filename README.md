# AI_Based_Movie_Recomendation_System
Overview
This project is a simple AI-based movie recommendation system that suggests movies to users based on their preferences, mood, and past ratings. It uses rule-based logic and basic natural language processing to understand user taste and recommend the most suitable movies.

Problem Statement
With the vast number of movies available today, users often struggle to find content that matches their interests. This project aims to simplify movie selection by analyzing user preferences such as favorite genres, mood, and ratings to provide personalized recommendations.

Solution
The system collects user input including favorite genres, current mood, and ratings for a few movies. It then builds a user profile using scoring logic and keyword extraction from movie descriptions. Based on this profile, it calculates scores for each movie and recommends the top matches.

The system also explains why each movie is recommended, making it more interactive and user-friendly.

Features
 - Personalized movie recommendations
 - User preference learning through ratings
 - Mood-based suggestions
 - Keyword-based matching from movie descriptions
 - Scoring system for ranking movies
 - Explanation for each recommendation
 - Technologies Used
 - Python (No external libraries)
 - Rule-Based AI Logic
 - Basic Text Processing
